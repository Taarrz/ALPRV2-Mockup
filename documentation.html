API Documentation

POST http://api.alprs/example

This endpoint allows you to send an image or video to the ALPR service and receive recognized license plate information in JSON format. You must include a valid API token in the request headers.

Example request:
curl -X POST http://api.alprs/example \
  -H "Authorization: Bearer <your_token>" \
  -F "file=@car.jpg"
